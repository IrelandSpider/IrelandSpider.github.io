config chinese charaters on build window at android studio

2020-10-11
  • precondition: windows10 android studio 4.0.2
  • situation: 使用android studio 进行编译项目，build 窗口有报错，但是是乱码
  • task: 在 android studio 双击shift然后进行搜索custom vm options，然后重启android studio,会生成一个文件studio64.exe.vmoptions,在里面添加一行-Dfile.encoding=UTF-8
  • result:正确显示中文
